FILES
-----------------------------------------------------------
Disk BASIC

[ 機能 ]
ディスク上のファイル・ディレクトリ一覧を表示します。

[ 書式 ]
FILES<ファイル名><,L>

[ 文例 ]
FILES "*.*",L

[ 解説 ]
<ファイル名> にはドライブ、
Disk BASIC 2 以降ではパスを含むことができます。
またワイルドカードで指定も可能です。
<,L> は Disk BASIC 2 以降で有効です。
ロングフォーマット形式により表示を行ないます。
