RUN
-----------------------------------------------------------
[ 機能 ]
フロッピーディスクからファイルを読み込み、そのプログラムを実行します。

[ 書式 ]
RUN <ファイルスペック>[,R]

[ 文例 ]
RUN "GAME.BAS"

[ 解説 ]
<ファイルスペック> を指定した場合は
フロッピーディスクからプログラムを読み込んだ後に
そのプログラムの実行を開始します。

初期値では開いているファイルが閉じられ、
メモリ内容はクリアされますが、
,R を指定するとファイルは開いたままで読み込まれます。
