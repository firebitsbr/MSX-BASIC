SAVE
-----------------------------------------------------------
[ 機能 ]
プログラムをファイルに保存します。

[ 書式 ]
SAVE <ファイル名>[,A]

[ 文例 ]
SAVE "TEST.BAS"

[ 解説 ]
<ファイル名> にはドライブ名、
DIsk BASIC 2 以降ではディレクトリ名を含む事ができます。

,A を付けた場合はアスキーモードで保存します。
アスキーモードは中間コードに変換されずに
入力されたプログラムの状態でそのまま保存されます。
MERGE 文を用いてプログラムの連結が可能なほ他、
テキストエディタ等で直接参照する事ができます。
