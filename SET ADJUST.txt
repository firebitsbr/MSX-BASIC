SET ADJUST
-----------------------------------------------------------
MSX2 以降

[ 機能 ]
画面の表示位置を調整します。

[ 書式 ]
SET ADJUST(<X座標>,<Y座標>)

[ 文例 ]
SET BEEP -1,1

[ 解説 ]
MSX 本体をテレビを用いて表示した際に
スクリーンモードによっては切れてしまう場合があります。
それを補正する機能です。
<X座標> <Y座標> はそれぞれ -8～7 の値です。
この設定は MSX 本体で SRAM に記録されます。
