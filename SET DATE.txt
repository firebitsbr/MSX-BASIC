SET DATE
-----------------------------------------------------------
MSX2 以降
MSXPLAYer では動作しません

[ 機能 ]
日付を設定します。

[ 書式 ]
SET DATE <日付>[,A]

[ 文例 ]
SET TIME "03-06-27"

[ 解説 ]
<日付> は <年>-<月>-<日> で、
<年> <月> <日> はそれぞれ2桁で指定します。
また、<年> は西暦の下2桁となります。
2003年2月1日であれば 03/02/01 となります。
,A をつけた場合はアラーム時間の日付設定を行ないます。
MSXPLAYer では使用している Windows の日付がそのまま使われます。
