STRIG
-----------------------------------------------------------
[ 機能 ]
スペースキー・ジョイスティックボタンの状態を得ます。

[ 書式 ]
STRIG(<番号>)

[ 文例 ]
A=STRIG(1):B=STRIG(3)

[ 解説 ]
<番号> は以下とおりです。

    0 スペースキー
    1 ポート 1 の A ボタン
    2 ポート 2 の A ボタン
    3 ポート 1 の B ボタン
    4 ポート 2 の B ボタン

押されていれば -1、押されていなければ 0 を返します。
