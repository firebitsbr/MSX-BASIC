WIDTH
-----------------------------------------------------------
[ 機能 ]
1行に表示する桁数を指定します。

[ 書式 ]
WIDTH<桁数>

[ 文例 ]
WIDTH 32

[ 解説 ]
SCREEN 0 では 1～40(MSX(1))・1～80(MSX2 以降)、
SCREEN 1 では 1～32 が指定できます。
初期設定では SCREEN 0 は 39 文字、
SCREEN 1 は 29 文字が指定されています。

漢字モードの場合、値は半角の文字数で、
CALL KANJI(0)・CALL KANIJ2 の場合は 26～64、
CALL KANJI1・CALL KANJI3 の場合は 26～80 が指定できます。
